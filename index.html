Adobe Photoshop

Uso

Una conocida aplicación de edición de imágenes y elaboración de contenido visual, útil para el diseño gráfico y el retocado fotográfico, producto de la empresa Adobe Inc.

Caracteristicas

Selección: esta es la característica más básica pero potente de Photoshop.
Capas – Capas es un conjunto de diferentes elementos apilados que contiene diferentes partes de una imagen. Esta es una de las mejores características creadas por Photoshop. Alteración del tamaño de la imagen: esta función amplía o reduce una imagen según sea necesario.
Recortar: recortar le permite seleccionar una parte de la imagen que desea conservar.
Afilado y suavizado: Photoshop tiene el poder de generar un efecto más nítido o borroso en las imágenes.
Fusión de imágenes: esta función permite que dos o más imágenes separadas se fusionen como una sola.
Eliminación de elementos no deseados: esta técnica implica el uso de la herramienta de clonación para mantener la textura de la imagen.
Cambio de color selectivo: el uso de esta función le permite modificar el color de ciertas partes de una imagen, manteniendo el resto de los colores originales.
Rebanado: ofrece la capacidad de cortar una imagen completa en partes y se etiqueta con diferentes archivos y luego se usa como contenido de un sitio web.
